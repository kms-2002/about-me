## 🚀 Key Projects & Experiences

* **2026.04 ~ 현재 | Global Pioneer Program (GPP) - 교내 해외 탐방 프로그램**
  * 국제통상, 스마트유통물류, 전자공학 등 다양한 전공자로 구성된 4인 융합 팀 프로젝트. 글로벌 전사적 흐름을 도출하고 비즈니스 모델을 구체화하는 다학제간 협업 수행.
* **2025.12 ~ 2026.03 | 경상남도 빅데이터 분석 공모전 출전**
  * 경상남도 지역 내 공공 데이터 및 다양한 도메인 데이터를 분석하여 지역 현안 해결 및 비즈니스 인사이트 도출을 목표로 한 공모전 참여.
  * **주요 수행 역할:**
    * **데이터 수집 및 전처리:** Python(Pandas, NumPy) 라이브러리를 활용하여 공공 데이터 포털 및 지자체 개방 데이터의 결측치 처리, 정규화, 파생변수 생성 등 데이터 가공 프로세스 전반 수행.
    * **통계 분석 및 시각화:** 분석 목적에 맞춘 다각적 통계 모델링과 Matplotlib, Seaborn 등을 활용한 시각화를 통해 데이터 분석 결과의 직관성과 가시성 극대화.
    * **비즈니스 모델 및 정책 제안:** 분석된 데이터 인사이트를 바탕으로 경상남도 지역 프로세스 혁신 및 실현 가능한 최적의 비즈니스 아키텍처 모델 기획·수립.
* **2025.06 | 거제시 세대별 관광지 선호도 데이터 분석 프로젝트**
  * Python과 Folium 라이브러리를 활용해 관광 데이터를 지리정보로 시각화. 공공 데이터를 바탕으로 세대별 맞춤형 관광 코스 제안 및 인사이트 도출.
* **2024.12 | 호텔 숙박 업소 예약 시스템 'HOTELLY' 웹 서비스 기획 및 개발**
  * 연말(11월~12월) 호캉스 및 여행지 주변 숙박 수요 증가 추세를 반영한 사용자 맞춤형 호텔 예약 플랫폼 구축 프로젝트.
  * **주요 기능 및 시스템 설계:**
    * **동적 캘린더 아키텍처:** JavaScript(Date 객체)를 활용해 실시간 달력 렌더링 함수(`renderCalendar`) 및 월 변경 로직을 설계하고, 클릭 시 성수기(12, 1, 2, 7, 8월)와 비수기를 자동 판별하는 알림 기능 구현.
    * **인터랙티브 UI/UX:** 메인 페이지 내 3초 주기 자동 이미지 슬라이드 배너, 마우스 오버(Hover) 시 반응하는 내비게이션 바, 이벤트 이미지 확대 기능, 예약하기 클릭 시 캘린더 서브 페이지 연동 설계.
    * **웹 컴포넌트 구성:** 메인/서브 페이지 UI 기획, 가입 및 로그인 프로세스 창 이동 설계, 상세 정보(체크인/체크아웃, 인원수 선택)가 연동되는 예약 레이아웃 구현.
